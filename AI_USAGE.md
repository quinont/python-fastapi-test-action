# AI Usage

Para trabajar con este repositorio estuve ocupando los siguientes LLM:

- **Gemini 3 Pro (chat):**
  - Lo ocupé básicamente para poder trabajar la idea; es decir, planteé algunas soluciones y las fui refinando con este LLM. La idea fue tenerlo como un par que me ayudara a pensar y a validar mis ideas.
  - Los *prompts* que usé fueron de varios tipos. Yo tengo más experiencia en GitLab y Jenkins, por lo tanto, las preguntas eran del tipo: "¿Cómo puedo hacer esto en GitHub Actions?" o "¿Cuál es la mejor forma de hacer esto en GitHub Actions?".
  - No tengo un solo *prompt* porque trabajé de varias formas y con chats muy extensos.

- **Jules (bot):**
  - Lo ocupé principalmente para que agregue los tests en el código. La idea era que yo pudiera estar trabajando en cómo solucionar los *pipelines* mientras que Jules avanzaba en la parte de programación y armado del código.
  - Los *prompts* que usé fueron del tipo: "Necesito que agregues tests unitarios y funcionales para el repositorio".

- **Antigravity (Gemini 3.1 Pro, Gemini 3 Flash, Claude 4.6 Sonnet):**
  - Lo ocupé para que me ayude a armar el código en sí, tanto de Kubernetes, Terraform como GitHub.
  - Los *prompts* fueron de muchos tipos, pero más yendo a la cuestión de no tomar decisiones por mí; o sea: "Necesito un módulo de Terraform que genere un Cloud SQL de tipo Postgres" o "Ayúdame a mejorar lo que dice este Markdown".

## Partes en las que influyó:

- Literalmente en todo: Terraform, GitHub Actions, Kubernetes, Python, Markdown, etc.
- Todos los textos que me fue devolviendo los fui probando y refinando con el LLM o yo mismo.
- La gran mayoría de las cosas que están aquí las fui probando y trabajando hace varios años. Lo que es nuevo para mí fue GitHub Actions, que gracias a las distintas analogías que hice con las herramientas que sí conocía, más la ayuda del LLM, pude avanzar.
